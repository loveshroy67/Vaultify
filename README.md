# Vaultify – AI-Powered Decentralized Data Locker

A decentralized application to securely store personal data using Solana blockchain.

## Features
- Encrypted data storage
- Decentralized architecture (Solana)
- Organized data categories
- Fast access

## Tech Stack
- Rust
- Solana
- React
- TypeScript

## Description
Designed modular backend components for secure on-chain data storage and retrieval.

Implemented encrypted transaction handling with optimized smart contract logic.

Integrated RESTful APIs for efficient client-server communication.

Structured storage mechanisms to improve retrieval efficiency.

Applied efficient data structures to manage structured on-chain storage and reduce redundant computations.
